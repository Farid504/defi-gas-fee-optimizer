# DeFi Gas Fee Optimizer ⛽

This repository logs average gas fees for popular networks and provides tips on reducing transaction costs.

## Chains Tracked
- Ethereum
- Solana
- Near
- Polygon

## Features
- Weekly average fee logs
- Tips for timing transactions
- Links to fee tracking tools
